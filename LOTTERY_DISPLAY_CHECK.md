# 彩票显示格式检查清单

## ✅ 已完成的彩种

### 1. 简单字符串格式
- ✅ **六合彩 (XGLHC, MOLHC)** - 6+1格式，圆形球，生肖颜色
  - 格式：`"8,47,10,39,5,19+45"`
  - 显示：6个圆形号码球 + "+" + 1个特码球
  - 带生肖和颜色波段
  
- ✅ **普通彩票** - 简单号码球
  - 格式：`"1,2,3,4,5"`
  - 显示：圆形红色渐变号码球

### 2. 复杂对象格式

#### ✅ 以太坊/区块链彩票
- 格式：`{code: "...", code_hash: "...", code_block: "..."}`
- 显示：号码球 + 区块信息
- 状态：**正常**

#### ✅ BAAC 泰国储蓄彩票
- 格式：`{code: "...", code0: "...", code1-8: ..., code_last3/4: ...}`
- 显示：简化列表 + "查看详情"按钮 + 模态框
- 状态：**正常**

#### ✅ MAX3D 特殊多段式
- 格式：`{code_1: "...", code_2: "...", code_3: "...", code_sp: "..."}`
- 显示：3组号码 + 特别号
- 状态：**正常**

#### 🔍 港式彩票（需检查）
- **六合彩 (XGLHC, MOLHC)** 
  - 格式：`{code: "8,47,10,39,5,19", code1: "45"}`
  - 显示：应该用6+1格式
  - 状态：**需要检查是否也用 + 号分隔**
  
- **其他港式彩票**
  - 格式：`{code: "...", code1: "..."}`
  - 显示：主要号码 + 附加号码
  - 状态：**正常**

#### ✅ 泰国政府彩票 (TGFC)
- 格式：`{code: "...", code1: "...", code2: "...", code3: "..."}`
- 显示：4个等级奖项
- 状态：**正常**

#### ✅ 带后缀号码彩票 (TLZC, YNHN, YNMA, TYKC)
- 格式：`{code: "...", code_last2/3: "...", code_pre2: "...", code_mid2: "..."}`
- 显示：主号码 + 后缀信息网格
- 状态：**正常**

#### ✅ ZCVIP 特殊格式
- 格式：`{code: "...", code2: "...", code_last2/3/4: "..."}`
- 显示：主号/副号 + 后缀网格
- 状态：**正常**

#### ✅ 越南传统彩票（41种）
- 格式：`{code: "...", code1-8: "..."}`
- 显示：简化列表 + "查看详情"按钮
- 状态：**正常**
